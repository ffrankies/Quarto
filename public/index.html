Quarto-Arena

Welcome to the Quarto Arena! This site is still in development.

Quarto is a fast-paced, two-player strategic board game, similar to tic-tac-toe. The game consists of a 4x4 game board, as well as a common pool of 16 unique pieces. The game objective of the game is to line up 4 pieces that have something in common.
